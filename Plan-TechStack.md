# 📚 ReadMate

> **개요:** 이미지 문서, PDF, 녹음 파일을 입력받아 로컬 AI 모델로 텍스트 인식(OCR), 요약, 퀴즈 생성을 제공하는 Streamlit 기반 학습 보조 도구

---

## 🛠️ 1. 로컬 기술 스택 (Local Tech Stack)

| 구분 | 선택 모델/도구 | 특징 및 비고 |
|------|--------------|------------|
| **UI Framework** | **Streamlit** | Python 기반 빠른 웹 인터페이스 구현 |
| **OCR Engine** | **Qwen2.5-VL-7B-Instruct** (4-bit NF4 양자화) | VLM 기반 한국어/영어 OCR. 복잡한 레이아웃 및 스캔형 PDF 처리 우수 |
| **Local LLM** | **Qwen2.5-7B-Instruct** | 로컬 요약, 퀴즈 생성, 쉬운 설명, 질의응답용 기본 모델 |
| **LLM 확장 후보** | **Qwen2.5-14B-Instruct** | 요약 품질과 맥락 유지가 더 필요할 때 업그레이드 |
| **API 대안** | **GPT-5.4-mini** | 더 빠른 프로토타이핑과 품질 확보가 필요할 때 사용 |
| **Environment** | **uv** + **Python** | 가상환경 및 패키지 관리 |

## 🔄 2. 전체 서비스 플로우 (Service Flow)

### Step 1: 입력 (Input)
- [ ] **이미지 획득:** Streamlit `st.camera_input` 또는 파일 업로드 기능을 통해 책 페이지 이미지 수신
- [ ] **전처리:** OpenCV를 활용해 이미지 회전 보정 및 노이즈 제거 (선택 사항)

### Step 2: 텍스트 추출 (OCR)
- [ ] **로컬 추론:** `Qwen2.5-VL-7B-Instruct` 모델이 이미지 내 한글/영문 텍스트 인식
- [ ] **데이터화:** 추출된 텍스트를 문맥에 맞게 하나의 문장/문단으로 병합

### Step 3: 지능형 분석 (Local LLM)
- [ ] **Context 전달:** 추출된 텍스트를 로컬에 실행 중인 `Qwen2.5-7B-Instruct` 모델로 전송
- [ ] **작업 수행:**
  - **요약:** 전체 내용을 핵심 3~5문장으로 요약
  - **퀴즈 생성:** 본문 기반의 객관식 또는 O/X 문제 2~3개 생성
  - **질의응답:** 학습자가 질문을 입력하면 본문 내용을 기반으로 답변 생성
  - **메모 가이드:** 학습자가 기록해야 할 핵심 키워드 도출
- [ ] **확장 방향:** 품질이 아쉬우면 `Qwen2.5-14B-Instruct`로 상향하거나, API 기반 `GPT-5.4-mini`로 전환

### Step 4: 학습 UI 출력 (Display)
- [ ] **결과 시각화:** 요약문, 퀴즈 인터페이스, 오디오 컨트롤러를 화면에 배치
- [ ] **상태 관리:** `st.session_state`를 활용해 퀴즈 정답 확인 및 다시 듣기 기능 지원
